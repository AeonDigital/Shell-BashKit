## Install
### Shrink version 

Open your shell (Bash compatible) and use:


``` shell
  curl -O "https://raw.githubusercontent.com/AeonDigital/Shell-BashKit/main/package-shell-bashkit.sh"

  # then, load
  chmod +x package-shell-bashkit.sh
  ./package-shell-bashkit.sh
```


### Full version

Clone the repository and load directly from the source:

``` shell
  # clone with
  git clone https://github.com/AeonDigital/Shell-BashKit.git

  # then, load
  cd Shell-BashKit
  chmod +x exec.sh
  ./exec.sh load
```

This loads all scripts from the `src/` folder in your current shell session, making all available functions ready to use.
